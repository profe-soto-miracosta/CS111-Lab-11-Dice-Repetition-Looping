# Lab 11 - Dice + Repetition/Looping

## Learning Objectives
- Use a for loop to repeat a task

## Program Description
Write a program which simulates a dice roll 100 times using a `for` loop, and keeps a count of how many times each number is rolled.

For every pass through the loop, generate a random number out of a set of 6 numbers (the number of sides on dice), and print the number rolled.

## Specifications
You have been given a method named `roll()` which will take care of the random number generation for you.  You just need to use a loop and variables to call the method 100 times, and then after 100 rolls of the dice, print out the counts of how many times each number was rolled.

## Sample Output

```
Roll 1: [3]
Roll 2: [1]
Roll 3: [6]
Roll 4: [4]
Roll 5: [2]
Roll 6: [3]
...
Roll 100: [5]

Final Roll Counts:
[1]: 15 times
[2]: 21 times
[3]: 14 times
[4]: 20 times
[5]: 16 times
[6]: 14 times
```
